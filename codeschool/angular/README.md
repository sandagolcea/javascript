#Angular Gemstore
Project for learning Angular
A static node server is used for serving ng-include files

#To run
* clone the repo (`git clone git@github.com:sandagolcea/javascript.git`)  
* `cd angular`  
* `npm install`  
* `node server.js`  
* goto localhost:3000
